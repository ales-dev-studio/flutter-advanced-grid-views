## **Quick Start**  
1. Clone the repo:  
   ```bash  
   git clone https://github.com/ales-dev-studio/flutter-advanced-grid-views.git 
   ```  
2. Install dependencies:  
   ```bash  
   flutter pub get  
   ```  
3. Run the app:  
   ```bash  
   flutter run  
   ```
